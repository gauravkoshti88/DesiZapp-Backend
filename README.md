# DesiZapp-Backend
DesiZapp – Food Delivery Website (MERN Stack)   A full‑stack food delivery platform built with MongoDB, Express.js, React.js, Node.js. Features include JWT authentication, Google OAuth login, real‑time order tracking with Socket.io, secure payments via Razorpay, and state management using Redux Toolkit.

Key Highlights -

Responsive UI with React + Tailwind CSS

Secure backend APIs with Node.js & Express.js

Authentication: JWT + Google OAuth

Real‑time order updates using Socket.io

Integrated Razorpay payment gateway

Redux Toolkit for scalable state management

Deployed on Render (backend) & Vercel (frontend)
